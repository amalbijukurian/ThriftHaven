# Thrift Haven Marketplace

A responsive website for a second-hand goods marketplace built using only HTML, CSS, and JavaScript.
Link: thrift-haven.vercel.app

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Interactive UI**: Modern and user-friendly interface
- **Category Browsing**: Browse items by different categories
- **Featured Items**: Showcase of featured second-hand items
- **Search Functionality**: Search for specific items
- **User Authentication**: Login and registration system (demo only)
- **Listing Creation**: Form to create new item listings
- **Testimonials**: Slider showing user testimonials
- **About Section**: Information about the marketplace
- **Newsletter Signup**: Form to subscribe to the newsletter

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript (ES6+)
- Font Awesome for icons

## Project Structure

- `index.html` - Main HTML file
- `styles.css` - CSS styles
- `script.js` - JavaScript functionality

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Navigate through the website using the menu
4. Try out the interactive features:
   - Search for items
   - Browse categories
   - View featured items
   - Create a listing (demo)
   - Login/Register (demo)

## Demo Features

This is a front-end only demonstration. The following features are simulated:

- User authentication (login/register)
- Item listing creation
- Search functionality
- Contact seller
- Newsletter signup

In a real-world application, these would be connected to a back-end server.

## Customization

You can customize this template by:

1. Modifying the color scheme in the CSS `:root` variables
2. Changing the featured items in the JavaScript `featuredItems` array
3. Updating the categories in the HTML
4. Adding your own images and content
